Temporal Raster Mean Plot

This operator generates a Temporal Raster Mean Plot based on a raster data set (Input). Select the Time Tick Position, which specifies the position of the x-axis ticks, e.g. at the the time start, end or center of the raster's validity. The plot shows an area chart of the mean values (y-axis) of the rasters. The x-axis displays the timestamps. Fill area under curve allows modifying whether to fill the area under the curve. Edit the Output Name of the plot, if desired, and click on Create to generate the plot.

Note: Plots change dynamically with the map view and include only pixels or features, which are located in this area.
